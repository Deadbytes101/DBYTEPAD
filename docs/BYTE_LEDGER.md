# Byte Ledger

DBYTEPAD measures the cost of the program instead of guessing.

Record format:

Version
Commit
Executable bytes
Executable SHA256
Source lines
Source bytes
Note

The ledger should be updated from a local build after the executable is produced.

Run:

```powershell
.\scripts\measure.ps1 -Record v0.1.4
```
