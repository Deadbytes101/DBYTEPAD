# DBYTEPAD v1.1.0

This release makes DBYTEPAD remember the way it was used.

It is still a small native Win32 editor written in C.

No Electron. No webview. No telemetry.

## Added

- Recent files.
- `dbytepad.ini` beside the executable.
- Window size memory.
- Font memory.
- Word Wrap memory.
- Font dialog.
- Replace.
- F5 Reload.
- Better unsaved-change prompt.

## Note

This release is not a rewrite.

It keeps the same Win32/RichEdit base and adds enough memory to make the editor feel like a tool instead of a demo.
