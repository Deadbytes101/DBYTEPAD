# DBYTEPAD v1.0.2

DBYTEPAD v1.0.2 is the stable public build.

It keeps the v1.0 editor behavior and adds the DBYTEPAD icon as a compact native resource.

## Build facts

- Executable bytes: 174080
- Source lines: 864
- Source bytes: 25190

## Included behavior

- Native Win32 window.
- RichEdit text buffer.
- Open, Save, Save As, Reload.
- Open Read Only.
- Find and Find Next.
- Word Wrap.
- Read Only mode.
- File Facts.
- Command-line file open.
- Dirty-state title marker.
- Line, column, char count, and UTF-8 byte count.
- Embedded DBYTEPAD icon.
- Byte Ledger measurement.

## Release note

No Electron.

No webview.

No runtime bundle.

No telemetry.

This release is the stable line. New feature work should happen away from `main` until it proves itself.
